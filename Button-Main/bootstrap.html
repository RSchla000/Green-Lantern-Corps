1. Text color classes (tiny rainbow of text)

These classes change text color quickly.

1).text-muted -- soft gray text (good for small notes)

2).text-primary -- bright blue text (for important content)

3).text-success -- soft green text (good for success messages)

4).text-info -- soft blue text (good for info messages)

5).text-warning -- soft yellow text (good for warnings)

6).text-danger -- soft red text (good for error messages)

7).text-secondary -- a darker gray text (good for less important content)

8).text-dark -- a darker text (good for main content)

9).text-white -- white text (I put it on a dark background so you can see it).

10).text-body -- default body text color
